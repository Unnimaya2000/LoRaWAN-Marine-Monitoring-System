LoRaWAN Marine Monitoring System

Overview

This project implements a MATLAB-based simulation of a LoRa communication system for marine monitoring.

Features

* LoRa chirp spread spectrum modulation
* Channel models (AWGN, Rayleigh, Rician)
* Hardware impairments (CFO, phase noise)
* BER vs SNR performance analysis

How to Run

Run the following scripts in order:

1. Chunk1_Setup
2. Chunk2_Modulation
3. Chunk3_Channel
4. Chunk4_BER_Decode
